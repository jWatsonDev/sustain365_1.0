# Sustain365

Sustain365 is a 6-week challenge tracker web app designed to help you build and maintain healthy habits. Track daily and weekly goals like prayer, reading, workouts, water intake, and more. Built with Next.js (App Router), Vercel Postgres, NextAuth.js, Resend, TypeScript, TailwindCSS, and PWA support for a seamless, modern experience.

## Features
- Daily habit tracking (prayer, reading, workout, water, food)
- Weekly goals (movement, cheat day/meal, flex week)
- Email-based authentication (magic link)
- Responsive, mobile-friendly UI
- Offline support (PWA)
- Hosted and auto-scaled on Vercel

## Tech Stack
- Next.js (App Router)
- Vercel Postgres
- NextAuth.js (Resend email provider)
- TypeScript
- TailwindCSS
- Prisma ORM
- PWA (manifest + service worker)
